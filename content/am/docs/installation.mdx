# ጊትን በመጠቀም

```bash
git clone https://github.com/ermi1999/ahadu.git && cd ahadu
```

# የ አሃዱ አጠቃቀም
አሃዱን ለመጻፍ በሁለት መንገድ መጀመር ይቻላል። አንደኛው በ መስተጋብራዊ ሼል

```bash
./ahadu
> አውጣ "አበበ በሶ በላ።";
አበበ በሶ በላ።
>
```

እና ሌላኛው ደግሞ የ አሃዱ ፋይልን በማሳለፍ ነው

```bash
./ahadu የፋይሉ_ስም.ah
```

# የኮድ መጻፊያን ለአሃዱ ማዘጋጀት

የኮድ መጻፊያዎትን ለአሃዱ ማዘጋጀትዎ የአሃዱን ቁልፍ ቃሎች የኮድ እዲተርዎ እንዲያውቅ እና እንዲያደምቃቸው ያደርጋል።

## Visual studio code

አሃዱ እራሱን የቻለ የvisual studio code ቅጥያ አለው። የሚከተሉትን መንገዶች በመጠቀም መጫን ይችላሉ።
