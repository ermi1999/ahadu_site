# ማጠር

አጥር ውስጥ ያሉ የ ኮድ ገለጻዎች በቅደም ተከተል የሚተገበሩ ዝርዝር የ ኮድ ገለጻዎች ናቸው። የኮድ ገለጻዎችን በ `{}` ቁልፍ ቃል ማጠር ይቻላል።

በ አጥር ውስጥ የተሰየሙ ማንኛውንም ስያሜዎችን ከ አጥሩ እይታ ውጪ ማግኘት አይቻልም። ይህም ማለት


```ahadu
መለያ ውጪያዊ_መለያ = 5;
{
    መለያ ውስጣዊ_መለያ = ውጪያዊ_መለያ + 10; // ውጪያዊ መለያን ከ ውስጥ ሆኜ ማግኘት እችላለሁ ውጤቱም 15 ይሆናል
}
ውጪያዊ_መለያ = ውስጣዊ_መለያ; // [!code error] // ነገር ግን ውስጣዊ መለያን ከ ውጪ ሆኜ ማግኘት አልችልም ውስጣዊ መለያ ከ እይታ ውጪ ነው
```

ውስጣዊ መለያን ማግኘት የምችለው በተመሳሳይ እይታ ወይም ከሱ በታች ካለ እይታ ውስጥ ሆኜ ብቻ ነው

```ahadu
{
    መለያ ውስጣዊ_መለያ = 5;
    {
        ውስጣዊ_መለያ = ውስጣዊ_መለያ + 1; // 6
        {
            ውስጣዊ_መለያ = ውስጣዊ_መለያ + 1; // 7
        }
    }
}
```

# ማጣመር

የማጣመር ኦፕሬተር `()` የ ኮድ ገለጻዎችን በማጣመር ቅድሚያ አሰጣጥ ሁኔታውን ለመቆጣጠር ያገለግላል

ይህንን ምሳሌ የመልከቱ

```ahadu
መለያ ሀ = 1;
መለያ ለ = 2;
መለያ ሐ = 3;

// ነባር ቅድሚያ አሰጣጥ
አውጣ ሀ + ለ * ሐ; // 7 ብሎ ያወጣል
// በነባር እንደዚህ ያየዋል
አውጣ ሀ + (ለ * ሐ); // 7 ብሎ ያወጣል

// አሁን ቅድመ አሰጣጡን በመቆጣጠር
// ከ ማባዛት በፊት መደመር
አውጣ (ሀ + ለ) * ሐ; // 9 ብሎ ያወጣል
```
