# ከሆነ ወይም ካልሆነ

ከሆነ ካልሆነ ገለጻዎች በሁኔታዎች ላይ ተመርኩዞ ድርጊቶችን ለመፈጸም ይረዱናል።

- <Link className="text-primary hover:text-off-blue transition-colors" href='/en/docs/the-language/conditional-statements#ከሆነ'>ከሆነን</Link> ሁኔታው እውነት ከሆነ አጥር ውስጥ ያለውን የኮድ ገለጻ ለመፈጸም እንጠቅምበታለን።
- <Link className="text-primary hover:text-off-blue transition-colors" href='/en/docs/the-language/conditional-statements#ከሆነ'>ካልሆነን</Link> ሁኔታው ሀሰት ከሆነ አጥር ውስጥ ያለውን የኮድ ገለጻ ለመፈጸም እንጠቅምበታለን።

## ከሆነ

```ahadu
ከሆነ (ሁኔታ) {
    // ሁኔታው እውነት ከሆነ የሚፈጸም ኮድ
}
```
ሁኔታ ውጤቱ እውነት ወይም ሀሰት የሆነ የኮድ ገለጻ። የትኛውም አውንታዊ ቁጥር እውነት ሲሆን እና የትኛውም አሉታዊ ቁጥር ሀሰት ነው። በተጨማሪም `ባዶ` እና `ሀሰት` ሀሰት የሆነ ውጤት ያመጣሉ ይህም ማለት በ አጥር ውስጥ ያለው ኮድ አይፈጸምም ማለት ነው።

```ahadu
ከሆነ (እውነት) {
    አውጣ "ይህ ይፈጸማል።";
}

ከሆነ (ሀሰት) {
    አውጣ "የህ አይፈጸምም።";
}
```

## ካልሆነ

ካልሆነን የ`ከሆነ` ሁኔታ ሀሰት ሲሆን እንዲፈጸም የምንፈልገውን የ ኮድ ገለጻ ለመግለጽ እንጠቅምበታለን።

```ahadu
ከሆነ (ሀሰት) {
    አውጣ "ይህ አይፈጸምም።";
} ካልሆነ {
    አውጣ "ይህ ይፈጸማል።";
}

ከሆነ (እውነት) {
    አውጣ "ይህ ይፈጸማል።";
} ካልሆነ {
    አውጣ "ይህ አይፈጸምም።";
}
```
