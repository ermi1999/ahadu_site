---
title: ለዚህ ተመላላሽ
description: ለዚህ ተመላላሽ በ አጥር ውስጥ ያሉ የ ኮድ ገለጻዎችን ለተወሰነ ጊዜ ለመደጋገም ይጠቅማል
---

ለዚህ ተመላላሽ በ አጥር ውስጥ ያሉትን የ ኮድ ገለጻዎችን ቅድመ ሁኔታው እውነት እስከሆነ ድረስ በተደጋጋሚ ይፈጽማል።

```ahadu
ለዚህ (መለያ = የኮድ_ገለጻ; ሁኔታ; መጨመር) {
    // የኮድ ገለጻዎች
}
```

## የለዚህ መለያ

በ መለያው ክፍል አዲስ መለያ መሰየም ይቻላል

```ahadu
ለዚህ (መለያ ሀ = 0; ሁኔታ; መጨመር) {
    // የኮድ ገለጻዎች
}
```

ወይም ደግሞ የተሰየመ መለያ መጠቀምም ይቻላል

```ahadu
መለያ ሀ; // [!code highlight]
ለዚህ (ሀ = 0; ሁኔታ; መጨመር) {
    // የኮድ ገለጻዎች
}
```

ወይም ደግሞ ባዶውንም መተው ይቻላል ነገር ግን ሴሚኮሎኑን ማስገባት አለብን። የ መለያ ስየማው ማብቃቱን ለመጠቆም

```ahadu
ለዚህ (; ሁኔታ; መጨመር) {
    // የኮድ ገለጻዎች
}
```

## የለዚህ ሁኔታ

የለዚህ ሁኔታ የተመላላሹን ማስቆሚያ ቅድመ ሁኔታ ለማስቀመጥ ይረዳናል። ይህም ማለት ተመላላሹ ቅድመ ሁኔታው እውነት እስከሆነ ድረስ ብቻ ነው የሚመላለሰው።

<Warning>
ሁኔታውን ባዶውን መተው ማለቂያ የሌለው ተመላላሽ ስለሚያስከትል ማለቂያ የሌለው ተመላላሽ ሲፈልጉ ብቻ ይተዉት።
</Warning>

```ahadu
ለዚህ (መለያ ሀ = 0; ሀ < 10; መጨመር) {
    // የኮድ ገለጻዎች
}
```

ይህ ኮድ ሀ ከ 10 በታች እስከሆነ ድረስ ብቻ ነው የሚመላለሰው።

<Info>
ለምን ያህል ጊዜ ተመላላሹ መመላለስ እንዳለበት አስቀድሞ የማይታወቅ ከሆነ <Link className="text-primary hover:text-off-blue transition-colors" href="/en/docs/the-language/while-statement">እስከ</Link> ተመላላሽን ቢጠቀሙ ይመከራል።
</Info>

## ለዚህ መጨመር
የለዚህ መጨመር ክፍል ከያንዳንዱ ምልልሶሽ በሗላ የሚተገበር ክፍል ሲሆን ይህም የት በ ተመላላሹ ውስጥ የት እንዳለን ለማወቅ የተተቀምነውን እሴት ለማሳደግ ይረዳናል።

```ahadu
ለዚህ (መለያ ሀ = 0; ሀ < 10; ሀ = ሀ + 1) {
    // የኮድ ገለጻዎች
}
```

ይህ ኮድ በእያንዳንዱ ምልልሶሽ ሀን ያሳድጋል ይህም ምልልሶሹ 10 ጊዜ ብቻ እንዲከናወን ያደርገዋል ማለት ነው።
