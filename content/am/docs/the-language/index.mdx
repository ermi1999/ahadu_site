# የአጻጻፍ ሁኔታዎች

አሃዱ ያ አብዛኞቻችን የምናውቀውን የC ን አይነት የአጻጻፍ ስልት ይከተላል። ስለዚህ ቶሎ ለመረዳት ቀላል ይሆንልዎታል። በዛ ላይ ደግሞ ነጻ የሆነ ቋንቋ ነው በ ቃላቶች መካከል የሚከሰቱ ባዶ ቦታዎችን ይዘላል።

የ አሃዱ ስሞች ሁሉም አይነት የላቲን ፊደሎች እና አማረኛ ፊደሎች መሆን ይችላሉ። በቁጥር እስካልጀመሩ ድረስ እና የተያዙ ቃላቶች እስካልሆኑ ድረስ። የ አሃዱ ስሞች መለያዎችን, ክፍሎችን እና ተግባሮችን ለመሰየም ያገለግሉናል።
