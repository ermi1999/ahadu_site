## አጻጻፉ

```ahadu
እስከ (ሁኔታ) {
    // የ ኮድ ገለጻዎች
}
```

### የእስከ ሁኔታ

የእስከ ሁኔታ ከለዚህ ሁኔታ ጋር ተመሳሳይ ሲሆን የተመላላሹን ማስቆሚያ ቅድመ ሁኔታ ለማስቀመጥ ይረዳናል። ይህም ማለት ተመላላሹ ቅድመ ሁኔታው እውነት እስከሆነ ድረስ ብቻ ነው የሚመላለሰው።

## ምሳሌ

የሚከተለው የለዚህ ተመላላሽ ሀ ከ10 እስካነሰ ድረስ በመመላለስ በ እያንዳንዱ ምልልሶሽ ደግሞ አሁን ላይ ሀ ያለውን እሴት ያወጣል።

ልብ ይበሉ ሀን በእያንዳንዱ ምልልሶሽ በ1 እያሳደግኩ ነው። ያንን ያላደረግኩ እንደሆን ማለቂያ የሌለው ተመላላሽ ይፈጠራል።

```ahadu
መለያ ሀ = 0;

እስከ (ሀ < 10) {
    አውጣ ሀ;
    ሀ = ሀ + 1;
}
```
