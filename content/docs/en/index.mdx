# What is Ahadu

Ahadu is a powerful, efficent, lightweight scrippting language. ahadu supports functional programming and object-oriented programming.

Ahadu is dynamically typed, runs by interpreting bytecode with a register-based virtual machine, and has automatic memory management with a generational garbage collection

## This is ahadu

Ahadu is written in clean C.
