This is the documentation for ahadu programming language.

# What is Ahadu

Ahadu is a powerful, efficent, lightweight scrippting language. ahadu supports functional programming and object-oriented programming.

Ahadu is dynamically typed, runs by interpreting bytecode with a register-based virtual machine, and has automatic memory management with a generational garbage collection, making it ideal for configuration, scripting, and rapid prototyping.

Ahadu is written in clean C.
